# Sample Insurance Policy PDFs

This directory is for storing uploaded insurance policy PDF files for processing by the chatbot.

## Instructions

1. Upload your insurance policy PDFs through the web interface
2. The files will be saved in this directory
3. The system will process them and extract the text for the knowledge base
4. You can then ask questions about the policies

## Supported File Types

Currently, only PDF files (.pdf) are supported.

## Privacy Note

The uploaded files are stored locally and are not sent to any external servers. The information is only processed within this application.
